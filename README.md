# CalcuSenha

Um jogo de adivinhação de senha numérica inspirado em Termo, desenvolvido em Python com interface gráfica usando Tkinter.

## Como Jogar

- O jogo tem uma senha secreta de 5 dígitos.
- Digite uma combinação de 5 números (0-9) usando os botões.
- Clique em "Verificar" para ver o feedback:
  - Verde: Número correto na posição certa.
  - Amarelo: Número correto, mas na posição errada.
  - Vermelho: Número incorreto.
- Continue tentando até acertar todos os 5 dígitos.

- ## Como rodar o jogo

- 1. Seu antivírus ou a própria proteção do SmartScreen do Windows vai apitar,
     dizendo que tem vírus. O jogo NÃO POSSUE VÍRUS, ele avisa por quê o jogo
     não tem uma licença (uma parada que os desenvolvedores pagam aí). Então é
     considerado malicioso.
  2. Após o susto, clica em "saiba mais" ou vai no antivírus e tira da quarentena,
     faz o que tiver que fazer aí. Daí deixa o jogo ser aceito para executar, depois
     é só jogar e se divertir, por uns 10 segundos ou uns 2 minuto, depende do seu
     racíocinio.
  3. Bom jogo!! :D
